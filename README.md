# AI Data Profiling Agent

An AI-powered data profiling application built using Streamlit, Pandas, and OpenAI.

## Features

- Upload CSV files
- Dataset profiling
- Null value analysis
- Duplicate detection
- Column statistics
- Interactive charts
- AI-generated insights

## Tech Stack

- Python
- Streamlit
- Pandas
- Plotly
- OpenAI API

## How to Run

```bash
streamlit run app.py
